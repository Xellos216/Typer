## Syntax

## 🔹 사용된 함수 설명
- `range(500)` → 최대 500번 반복하기 위한 범위 생성
- `num % 2 == 0` → 짝수 판단 조건
- `num // 2` → 정수 나눗셈
- `num * 3 + 1` → 홀수일 경우의 연산 규칙

### 🔹 사용된 문법 설명
- `for i in range(...)` → 최대 반복 횟수를 제한하기 위한 반복문
- `if ... else ...` → 짝수와 홀수 조건 분기 처리
- `return i`, `return -1` → 조건에 따라 반복 횟수 또는 실패 값을 반환
