## 🔹 `x % divisor == 0`
- 나머지가 0이면 나누어 떨어짐을 의미
- 예: `10 % 5 == 0` → True

## 🔹 `sorted()` 함수
- 리스트를 오름차순으로 정렬하여 새 리스트를 반환
- 예: `sorted([3, 1, 2])` → `[1, 2, 3]`

## 🔹 리스트 컴프리헨션
- `[x for x in arr if x % divisor == 0]` 
  → 조건을 만족하는 요소들만 추출하여 새 리스트 생성

## 🔹 삼항 조건 표현식
- `A if 조건 else B`
  조건이 참이면 A, 거짓이면 B 반환
- 예: `result if result else [-1]`
  → `result`가 비어 있지 않으면 반환, 비었으면 `[-1]` 반환
