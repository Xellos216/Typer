## 🔹 사용된 함수 및 문법 설명

### 🔹 `len()`
- 문자열의 길이를 구함  
  예: `len("hello")` → 5

### 🔹 인덱싱 vs 슬라이싱
- `s[i]` → i번째 문자 하나  
- `s[i:j]` → i 이상 j 미만 범위 슬라이싱

### 🔹 나눗셈과 몫
- `n // 2` → 정수 나눗셈, 가운데 위치를 계산할 때 사용

### 🔹 조건문
```python
if n % 2 == 0:
    ...
else:
    ...
```
- 문자열 길이의 짝/홀 여부에 따라 분기 처리
