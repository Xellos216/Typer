## 🔹 사용된 함수 및 문법 설명

### 🔹 `enumerate()`
- 문자열 또는 리스트를 반복할 때 인덱스와 값을 동시에 가져옴
- 예: `for i, c in enumerate("abc")` → i=0, c='a' ...

### 🔹 딕셔너리 (dict)
- 특정 문자의 마지막 위치를 저장하기 위한 자료구조
- `last_seen[char] = i` 와 같이 사용

### 🔹 거리 계산
- 같은 문자가 앞에 나왔는지 확인한 뒤,
  - 있다면 거리 계산: `i - last_seen[char]`
  - 없다면 `-1` 저장

---

## 🔍 예시

```python
s = "banana"

# 각 문자 처리
# b → 처음 등장 → -1
# a → 처음 등장 → -1
# n → 처음 등장 → -1
# a → 이전 위치 1 → 거리 3 - 1 = 2
# n → 이전 위치 2 → 거리 4 - 2 = 2
# a → 이전 위치 3 → 거리 5 - 3 = 2

# 결과: [-1, -1, -1, 2, 2, 2]
```
