## Syntax

## 🔹 사용된 문법 설명

### 🔹 조건문 `if-else`
- 두 값이 같은지 비교할 때 사용: `if num1 == num2`
- 같으면 `1`, 다르면 `-1` 반환

### 🔹 비교 연산자 `==`
- `a == b`: a와 b가 같으면 `True`, 아니면 `False`
- 예: `5 == 5` → `True`, `3 == 2` → `False`

### 🔹 return 문
- 조건에 따라 다른 값을 반환

---

## 🔍 예시

```python
num1 = 2
num2 = 3
# 2 != 3 → return -1

num1 = 11
num2 = 11
# 11 == 11 → return 1
```
