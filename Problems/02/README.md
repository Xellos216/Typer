## 🧠 Problem
정수 `num1`, `num2`가 주어질 때, `num1`과 `num2`를 곱한 값을 return하는 함수를 작성하시오.

### Constraints
- 0 ≤ `num1`, `num2` ≤ 100

### Example
- Input: `num1 = 3`, `num2 = 4`
- Output: `12`

- Input: `num1 = 27`, `num2 = 19`
- Output: `513`

---

## 🐍 Python Reference Code

```python
def solution(num1, num2):
    return num1 * num2
```
