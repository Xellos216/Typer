## 🔹 사용된 개념 및 문법 설명

### 🔹 3진법 변환
- `n % 3` → 현재 자릿수
- `n //= 3` → 다음 자릿수로 이동
- 거꾸로 누적하여 문자열로 저장

```python
while n:
    result = str(n % 3) + result
    n //= 3
```

---

### 🔹 문자열 뒤집기
- `s[::-1]` → 문자열을 뒤집음  
  예: `"1200"[::-1]` → `"0021"`

---

### 🔹 진법 변환 (3진수 → 10진수)
- `int(s, 3)` → 문자열 `s`를 3진수로 보고 10진수로 변환  
  예: `int("0021", 3)` → `7`
