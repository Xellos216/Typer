## 🧠 Problem
정수 `num1`과 `num2`가 주어질 때, `num1 + num2`의 합을 return하는 함수를 작성하시오.

### Constraints
- -50,000 ≤ `num1`, `num2` ≤ 50,000

### Example
- Input: `num1 = 2`, `num2 = 3`
- Output: `5`

- Input: `num1 = 100`, `num2 = 2`
- Output: `102`

---

## 🐍 Python Reference Code

```python
def solution(num1, num2):
    return num1 + num2
```
