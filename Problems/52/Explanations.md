## Syntax

## 🔹 사용된 함수 및 문법 설명

### 🔹 while 반복문
- `while n >= a`: 교환 조건이 만족될 때까지 계속 반복
- 조건이 False가 되면 루프 종료

### 🔹 몫 연산 (`//`), 나머지 연산 (`%`)
- `n // a`: 교환 가능한 세트 수
- `n % a`: 교환하고 남은 병 개수

### 🔹 변수 누적 방식
- `answer += exchanged`: 지금 교환한 콜라를 총량에 누적

---

## 🔍 예시 흐름

```python
a = 3, b = 1, n = 20

# Step 1: 20개 중 6세트 교환 → 콜라 6개, 빈 병 6개
# Step 2: 6개 중 2세트 교환 → 콜라 2개, 빈 병 2개
# Step 3: 3개 중 1세트 교환 → 콜라 1개, 빈 병 1개
# Step 4: 1개 < a → 더 이상 교환 불가 → 종료

# 총 콜라 = 6 + 2 + 1 = 9
```
