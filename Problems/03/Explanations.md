## Syntax

## 🔹 사용된 연산 설명

### 🔹 정수 나눗셈 연산자 `//`
- `a // b`는 a를 b로 나눈 **몫(정수 부분)**을 반환
- 예: `10 // 3` → `3`, `7 // 2` → `3`

### 🔹 return 문
- 함수의 결과를 반환하는 구문
- 예: `return num1 // num2`

---

## 🔍 예시

```python
num1 = 10
num2 = 5
# 10 // 5 = 2 → return 2

num1 = 7
num2 = 2
# 7 // 2 = 3 → return 3
```
