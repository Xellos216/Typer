## 🔹 사용된 함수 및 문법 설명

### 🔹 `range(start, end)`
- 반복 범위 지정
- `range(1, n+1)`은 1부터 n까지 반복

### 🔹 나머지 연산자 `%`
- `n % i == 0` → i가 n의 약수인지 확인

### 🔹 누적 덧셈
- 조건이 참이면 `answer += i`로 누적

---

## 🔍 예시

```python
n = 12
# 약수: 1, 2, 3, 4, 6, 12 → sum = 28 → return 28

n = 5
# 약수: 1, 5 → sum = 6 → return 6
```
