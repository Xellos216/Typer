## Syntax

## 🔹 사용된 함수 및 문법 설명

### 🔹 `range(2, n)`
- 2부터 n-1까지 반복 (자연수 조건 만족)

### 🔹 나머지 연산자 `%`
- `n % x == 1` 조건 확인 → x가 나머지가 1이 되는 수인지 검사

### 🔹 조기 반환 `return`
- 조건을 만족하는 x를 찾자마자 바로 반환 → 가장 작은 값 보장

---

## 🔍 예시

```python
n = 10
# 10 % 2 = 0 → x=2 불만족
# 10 % 3 = 1 → 조건 만족 → return 3

n = 12
# 12 % 2 ~ 10 → 나머지 ≠ 1
# 12 % 11 = 1 → return 11
```
