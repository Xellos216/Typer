## 🧠 Problem
문자열 `s`가 주어질 때, 해당 문자열을 정수로 변환한 결과를 반환하는 함수를 작성하시오.

### Constraints
- `s`의 길이는 1 이상 5 이하
- `s`는 부호(`+` 또는 `-`)와 숫자로만 구성됨
- `s`는 `"0"`으로 시작하지 않음

### Example
- Input: `"1234"`
- Output: `1234`

- Input: `"-1234"`
- Output: `-1234`

---

## 🐍 Python Reference Code

```python
def solution(s):
    return int(s)
```
