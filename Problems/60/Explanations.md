## Syntax

### 🔹 약수 개수 구하기

```python
for i in range(1, int(n**0.5) + 1):
    if n % i == 0:
        count += 1
        if i != n // i:
            count += 1
```

- 약수는 `√n`까지만 확인해도 충분
- `i`와 `n // i` 두 쌍이 약수 관계 (단, `i == n // i`인 경우 중복 제외)

---

## 🔍 주요 로직 흐름

1. `1`부터 `number`까지 모든 기사 번호에 대해 순회
2. 각 번호에 대해 약수 개수(공격력)를 구함
3. 공격력이 `limit`보다 크면 → `power`로 대체
4. 각 기사의 무기 공격력을 모두 합산
5. 무기 공격력 = 철의 무게(1kg당 1 공격력) → 총합을 그대로 리턴

---

### ✅ 예시 흐름 (number = 10, limit = 3, power = 2)

| 기사 번호 | 약수 개수 | 제한수치 비교 | 적용 공격력 |
|-----------|------------|----------------|--------------|
| 1         | 1          | ≤ 3            | 1            |
| 2         | 2          | ≤ 3            | 2            |
| 3         | 2          | ≤ 3            | 2            |
| 4         | 3          | ≤ 3            | 3            |
| 5         | 2          | ≤ 3            | 2            |
| 6         | 4          | > 3            | 2 (치환됨)   |
| 7         | 2          | ≤ 3            | 2            |
| 8         | 4          | > 3            | 2 (치환됨)   |
| 9         | 3          | ≤ 3            | 3            |
| 10        | 4          | > 3            | 2 (치환됨)   |

→ 총합 = 1+2+2+3+2+2+2+2+3+2 = `21`

## ✅ Summary

| 단계       | 설명                                     |
|------------|------------------------------------------|
| 약수 계산   | √n까지만 반복해 효율적으로 계산               |
| 제한 비교   | `limit` 초과 시 `power`로 치환               |
| 철 무게 계산 | 각 기사의 공격력 총합 = 필요한 철 무게(kg) 총합 |
